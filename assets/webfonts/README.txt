WEBFONTS IN THEME
- - - - - -

How to use webfonts?
1. Add your webfont files to /assets/webfonts/ folder
2. Add @font-face rules in /assets/css/_base.css (remember to set font-display rule)
3. Preload fonts in /inc/_conf/register-assets.php -> avidly_theme_preload_webfonts() function

That's it!
